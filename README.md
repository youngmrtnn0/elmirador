Qué tiene cada versión
v1.0.0 (Login)
Pantalla dividida a la mitad. Fondo azul a la izquierda con el logo y formulario blanco a la derecha. Abajo tiene el cuadro gris con los correos de prueba.

v1.0.1 (Dashboard)
Panel del administrador con los 4 cuadritos arriba (Torres: 4, Departamentos: 128, Bodegas: 64, Estacionamientos: 112), el gráfico redondo y la lista de actividad reciente.

v1.1.0 (Módulos de administración)
Se agregaron las pantallas para controlar todo: la ficha de la comunidad con sus pestañas, las torres divididas por plantas con colores (verde, gris y amarillo) y la lista de unidades con buscador y botones para ver o editar.

v1.2.0 (Portal del residente)
Vista para el vecino Andrés Lagos. Se sacó el menú de la izquierda para que se vea más limpio, tiene una barra azul arriba y la tarjeta con los datos de su departamento (U-301).

v2.0.0 (Correcciones y alertas)
Se armó la ventana para asignar bodegas y estacionamientos que muestra el aviso verde de éxito arriba a la derecha. También se agregó la campana de notificaciones para el residente y la lista de áreas comunes abajo en gris.
